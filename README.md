# floating_navigation_bar

A beautiful, highly customizable floating navigation bar for Flutter with smooth animations, badge support, and multiple styles (circular, pill, rectangular).

Perfect for modern apps that want a premium bottom navigation experience.

## Features

- Smooth slide & fade animations
- Badge support on navigation items
- 3 built-in styles: circular, pill-shaped, rectangular
